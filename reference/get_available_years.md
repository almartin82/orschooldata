# Get available years for Oregon enrollment data

Returns the range of years for which enrollment data is available.

## Usage

``` r
get_available_years()
```

## Value

Integer vector of available years

## Examples

``` r
get_available_years()
#>  [1] 2010 2011 2012 2013 2014 2015 2016 2017 2018 2019 2020 2021 2022 2023 2024
#> [16] 2025
```
